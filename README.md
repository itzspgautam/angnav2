## ANGNA
Angna is a web-based platform built for the Association of Garhwa Navodaya Alumni. It is designed for students of Jawahar Navodaya Vidyalaya who can participate in multiple events such as dance competitions, essay competitions, art and painting, mehandi design, etc. The platform enables students to upload their files and allows the alumni association to share results and updates through a blog.

## Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

## Prerequisites
You will need to have Node.js installed on your machine.

## Installing
Clone the repository: git clone https://github.com/itzspgautam/Angnav2.git
Change into the project directory: cd Angna
Install the dependencies: npm install
Running the project
To run the project locally, run the following command in the project directory: npm run dev
